# PaxRL

**PaxRL** is an efficient 2D simulator for Swarm Robotic applications written with [Jax](https://github.com/google/jax).

## Installation:

Run the command:
```
chmod +x install.sh
```
### CPU Version
Run the command:
```
install.sh cpu
```

### NVIDIA CUDA 12.2 Version
Run the command:
```
install.sh gpu
```
